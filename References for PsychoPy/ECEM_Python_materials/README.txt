Materials from the ECEM Python for Eye-tracking workshop
============================================================
Sol Simpson and Jon Peirce


Contained in this folder you will find the following:
    - html/index.html
        this is a good starting place for all the materials!
    - PyET.pdf
        this is a pdf equivalent of the above
    - demos
        working copies of the demos that were built in the workshop
    - python_source
        some scripts showing you how to do various analyses
    - IPython_notebooks
        for those that use IPython many of the materials were created in this format as well
    - PsychoPy-1.78.00 Manual.pdf
        This... ummm... you can work that one out!
